SQUAD-MAKER

나만의 스쿼드 짜기

  • 포메이션 변경 기능 구현 (4-3-3, 4-2-3-1, 4-1-2-3, 4-4-2, 5-2-3)
  • 나의 카드에서 만든 나만의 카드 선수 등록 기능 구현
  • 나의 카드에서 만든 나만의 카드 후보 등록 기능 구현
  • 세부사항 보기 버튼 클릭 후 선수나 후보를 누르면 그 선수의 세부사항을 볼 수 있음.
  • 만든 스쿼드 저장, 불러오기, 삭제 기능 구현

나만의 전술을 짤 수 있는 페이지

  • 포메이션 변경 기능 구현 (4-3-3, 4-2-3-1, 4-1-2-3, 4-4-2, 5-2-3)
  • 세트피스 상황을 구현할 수 있는 기능 구현 (ex. 코너킥)
  • 선수나 공을 직접 마우스로 드래그해 자유롭게 움직일 수 있는 기능 구현
  • 패스, 선수의 움직임, 슈팅을 선으로 시각화 할 수 있는 기능 구현
    • 패스 or 선수 움직임, 슈팅 버튼을 누른 후 선의 시작점, 끝점을 클릭하면 선이 생성됨.
    • 모든 선 삭제를 누르면 모든 선이 삭제되고 선 선택 삭제 버튼을 누르면 내가 원하는 선을 클릭하여 삭제 가능

나만의 카드를 만들 수 있는 페이지

  • 내가 만들 선수 카드의 이미지 선택 기능 구현 (선택하지 않을 시 기본 이미지 로드)
  • 이름, 등번호, 그 카드의 스텟을 직접 기입하는 기능 구현
